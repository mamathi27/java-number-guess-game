# 🎯 Number Guessing Game (Java)

A simple console-based Number Guessing Game developed using Java.  
The program generates a random number, and the user has to guess it with hints provided after each attempt.

---

## 🚀 Features

- Random number generation
- User input handling
- Feedback on each guess (Too High / Too Low)
- Continuous guessing until the correct number is found

---

## 🛠️ Technologies Used

- Java
- Core Java Concepts (Loops, Conditionals, Random, Scanner)

---

## 📌 How It Works

1. The program generates a random number within a given range.
2. The user is prompted to enter a guess.
3. The program compares the guess with the actual number.
4. It provides feedback:
   - "Too High" if the guess is greater
   - "Too Low" if the guess is smaller
5. The process continues until the correct number is guessed.

---

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/mamathi27/java-number-guess-game.git

2. Navigate to the project folder:
   ```bash
   cd java-number-guess-game
   ```
3. Compile the program:
   ```bash
   javac Main.java
   ```
4. Run the program:
   ```bash
   java Main
   ```
